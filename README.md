## Description
A performance overhaul for the **FS-20 Vortex**, transforming it from a capable multirole fighter into a true hypersonic superiority platform. Every core system has been upgraded — from the deck up.

## Features

### ⚡ Quadruple Thrust VTOL
Engine thrust is boosted to **400,000 N** (4× stock), with VTOL thrusters scaled to match. The Vortex can now lift vertically from any airfield or carrier regardless of payload weight — fully loaded, no excuses.

### 🔥 Scramjet Mode
Above **Mach 4.5 at 10,000 m**, the engine transitions into scramjet operation at **500,000 N**. Engage and hold altitude to sustain hypersonic cruise. Disengage drops back to baseline thrust with hysteresis to prevent flicker.

A pulsing **SCRAMJET ACTIVE** indicator appears on the HUD when the mode is live.

### 🌌 Operational Ceiling
Engine flames out at exactly **50,000 m**. Below that, the altitude thrust curve is flat — no power loss with altitude.

### 📡 Sensor Suite
- **Radar**: detection range and signal strength doubled
- **EOTS**: visual range extended to 50 km, 10× magnification, tracks targets up to 5,000 m/s

### 🔋 Power System
Capacitor capacity, recharge rate, and max power output each increased **10×** — sustained high-draw systems stay online indefinitely.

### 🛡️ Airframe Hardening
- Airbrake joints reinforced for hypersonic deployment
- FBW pitch/roll damped above Mach 3 to prevent oscillation
- Overspeed warnings suppressed up to **4,500 m/s**

## Compatibility
Does not modify the aircraft definition or encyclopedia entry — all existing liveries remain fully accessible. Compatible with QoL too.
